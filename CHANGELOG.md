## [0.1.0] - 2026-03-05

### Added

- Initial release of the Unique Names Data Package
- **Dictionaries:**
  - `adjectives.csv` — 120 general adjectives (personality, physical, abstract)
  - `colors.csv` — 50 color names
  - `animals.csv` — 100 animal species
  - `celestial.csv` — 50 space and astronomy terms
  - `nature.csv` — 50 natural elements and phenomena
  - `science.csv` — 50 scientific terms
  - `numbers.csv` — 100 numeric strings (0-99)
- **Moderation:**
  - `blocklist.csv` — Blocklist structure for excluded words
- **Documentation:**
  - `README.md` — Overview and usage examples
  - `CONTRIBUTING.md` — Word curation policy and contribution guidelines
  - `CHANGELOG.md` — Version history
- **Metadata:**
  - `datapackage.json` — Frictionless Data Package v2 descriptor
  - `LICENSE` — CC0-1.0 public domain dedication
- **CI/CD:**
  - GitHub Actions workflow for validation
